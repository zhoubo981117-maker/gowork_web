## 模块二：分析页面（新功能：图片粘贴）

### UAT-ANA-01 文本输入分析

| 步骤 | 操作 | 预期结果 | 实际结果 |
|------|------|---------|---------|
| 1 | 进入分析页面 | 显示 JD 和简历两个输入框 | |
| 2 | 在 JD 框输入一段职位描述（>20字） | 文本正常显示 | |
| 3 | 在简历框输入简历内容（>20字） | 文本正常显示 | |
| 4 | 点击"开始分析" | 显示匹配分数、差距分析、推荐建议 | |

### UAT-ANA-02 截图粘贴到 JD 框（新功能）

| 步骤 | 操作 | 预期结果 | 实际结果 |
|------|------|---------|---------|
| 1 | 对招聘页面截图（Snipping Tool 或 PrtScn）| 截图保存到剪贴板 | |
| 2 | 点击 JD 输入框，按 Ctrl+V | 显示"识别中..."加载状态 | |
| 3 | 等待 2-5 秒 | JD 框自动填入识别出的文字 | |

> ⚠️ 注意：OCR 功能需要服务端安装 tesseract-ocr。未安装时会提示"OCR服务不可用"，属正常行为。

### UAT-ANA-03 上传文件提取简历

| 步骤 | 操作 | 预期结果 | 实际结果 |

| 1 | 点击简历区域的"上传文件" | 弹出文件选择框 | |
| 2 | 选择一个 PDF 或 TXT 文件 | 显示"上传中..."，完成后简历框自动填入内容 | |
| 3 | 修改提取的文字 | 文字可编辑 | |

## 模块四：面试模拟（新功能：从看板选岗位）

### UAT-INT-01 从看板选择岗位开始面试（新功能）

**前置条件**: 看板中至少有一个职位（UAT-BOARD-01 已通过）

| 步骤 | 操作 | 预期结果 | 实际结果 |
|------|------|---------|---------|
| 1 | 进入面试模拟页面 | 显示"选择岗位"下拉框，列出看板中的职位 | |
| 2 | 从下拉框选择一个有职位描述的岗位 | JD 文本框自动填入职位描述 | |
| 3 | 查看简历文本框 | 若该岗位有上传简历，自动填入；否则显示⚠️提示 | |
| 4 | 点击"开始面试" | 进入面试对话界面，出现第一个问题 | |

### UAT-INT-02 手动输入开始面试

| 步骤 | 操作 | 预期结果 | 实际结果 |
|------|------|---------|---------|
| 1 | 进入面试模拟页面，不选择岗位 | JD 和简历框为空 | |
| 2 | 手动输入 JD 和简历内容 | 输入正常 | |
| 3 | 点击"开始面试" | 进入面试对话 | |

### UAT-INT-03 面试对话流程

| 步骤 | 操作 | 预期结果 | 实际结果 |
|------|------|---------|---------|
| 1 | 面试开始后，看到第一个问题 | 问题显示正常 | |
| 2 | 输入一个回答，提交 | 显示反馈（优点/不足/建议答案），出现下一个问题 | |
| 3 | 完成所有问题或点击"结束面试" | 显示面试总结和评分 | |
